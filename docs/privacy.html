Privacy Policy

Last updated: January 2026

This application is used to automate the posting of original video content created by the developer to social media platforms (TikTok, YouTube, Instagram).

Information We Collect

The application only accesses the data strictly necessary to publish content, such as authentication tokens and basic account information provided by the platform APIs.

How We Use Information

Information is used solely to authenticate the user and publish videos on the user's own accounts. No data is sold, shared, or used for advertising.

Data Storage

Authentication tokens are stored securely and are only accessible by the developer.

Third-Party Services

This application uses official APIs provided by TikTok, Google (YouTube), and Meta (Instagram).
